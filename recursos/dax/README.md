2. Gere os dados com `script.md`
3. Faça os exercícios em `exercicios.md` (básico)
4. Confira as resoluções em `resolucoes.md`
5. Avance para `exercicios-time-intelligence.md` (avançado)
6. Confira as resoluções em `resolucoes-time-intelligence.md`
